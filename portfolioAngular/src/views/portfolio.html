Steal My Deck

SpeakEasy
  • An App that allows users to meet up and exchange skills and languages. This is a full stack app with full CRUD functionality as well as user authentication. It was built as a team of four where we worked in an agile environment. It is built with Node.js and Angular, styled with sass and deployed with Heroku. We also incorporated the GoogleMaps API, the GooglePlaces API and the NodeMailer API.
  • Angular, Angular Material Design, JavaScript, ES6, CSS, HTML, BCrypt, Ajax, JSON, Express, Web Google Maps API, NODE.js, NodeMailer, Mongoose, Mongo, CSS, Bootstrap, Gulp
